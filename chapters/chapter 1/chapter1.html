INTRODUCTION AU MODULE TIC

1 - Les techniques de l'information et de la communication (TIC)

C'est l'ensemble des outiles disponible pour manipuler des information dans tous leur aspect (stocke ,traiter,produire,transmettre)

On peut citer ces utilisation

  • Communiquer et Transmettre (réseaux, ...)
  • Accéder aux sources d'informations (les navigateurs,)
  • Stocker (moyens de stockage)
  • Manipuler (les PC, tablettes, scanner)
  • Produire (imprimante, logiciels de production, ...etc.), sous forme :
    • Texte
    • Image Et interface graphique
    • Son
    • Vidéo
  • En utilisant les techniques de :
    • L'informatique et Internet
    • L'audiovisuel
    • Multimédia

2 - Catégories et objectifs

Les TIC sont présentés dans 3 grandes categories :

  • 1-La téléinformatique :

    • C'est la science qui permet d'echanger des information numerique entre systeme informatique

  • 2-Télécommunication :

    • C'est le domaine ou les systeme communicats ne sont pas digitaliser

  • 3-L'électronique :

    • Les composants comme TV ORDINATEUR LECTEUR CD

Objectifs recherches :

  • La connectivite
  • La modularite
  • Une implantation simple
  • Une utilisation facile
  • La fiabilité

Objectifs de communication

Un objectif de communication doit se définir selon 3 dimensions

  • Cognitif : la communication a pour but La prise de conscience
  • Affectif : c’est de Modifier ou renforcer l’image de produit
  • comportemental : » le but est de modifier les comportements en provoquant l'achat

Systeme de communication

Hello, again!

en general il contient les conspect suivant

  • Les partenaires de communication
    • L’émetteur : il transforme le message en signaux.
    • Le récepteur : il reçoit les signaux et reconstruit le message.
  • Les vecteurs
    • Message : l’émetteur émet un message qu’il va tenter de coder le plus parfaitement possible
    • Médias : c’est le moyen utilisé, pour transmettre le message.
  • Les fonctions
    • Codage : c’est la traduction du message en un ensemble structuré de signifiants
    • Décodage : c’est l’interprétation du récepteur
    • Réponse : l’avis du client
    • Feedback : c’est la réaction du récepteur face au message reçu
  • Bruit : il peut venir perturber les signaux pendant la transmission

Les types de communication

  • La communication institutionnelle (corporate)
  • La communication commerciale ou de produit

Les schemas de communication

  • La communication interpersonnelle : Sera employée lorsque vous allez vous adresser à l’un de voscollaborateurs
  • La communication de groupe : Est le schéma utilisé lorsque vous vous adressez à vos salaries
  • La communication de masse : est par exemple celle de la propagande

Média et Multimedia

Media

Un média est un moyen impersonnel permettant une diffusion large et collective d'informations, quel que soit le support

  • Les messages
  • La radiodiffusion
  • La télédiffusion
  • Le réseau Internet

Multimedia

Ensemble des techniques et des produits qui permettent l'utilisation simultanée et interactive de plusieurs modes de représentation de l'information
