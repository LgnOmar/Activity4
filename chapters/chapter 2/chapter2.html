Les réseaux informatiques

1 - Réseau (informatique) Network

Les réseaux informatiques permettent l'échange de données et de ressources. Ils sont utilisés dans de nombreuses applications telles que la communication, la gestion de données, les réseaux sociaux, l'accès à Internet, les jeux en ligne, etc. Cependant, ils présentent des risques pour la sécurité tels que le piratage et les cyberattaques. La sécurité des réseaux et des données est donc essentielle.

-En utilisant le modèle OSI, les fabricants de matériel et de logiciels peuvent concevoir des produits compatibles avec les différents composants d'un réseau informatique, assurant ainsi une interopérabilité entre les équipements de différents fournisseurs.

2 - Les caractéristiques de base d’un réseau sont

  • La topologie
  • Le débit exprimé en bits/s (ou bps)
  • La distance maximale (ou portée)
  • Le nombre de nœuds maximum

6 - Le type de topologie

Le type de topologie (architecture matérielle et/ou logique) il existe (bus , anneau , étoile , maillée) :

  • Bus Network

    • Les réseaux en bus:

    Les réseaux en bus tels que Ethernet et Token-Bus sont des types de réseaux informatiques où tous les périphériques sont connectés à une même ligne de communication appelée le bus. Les données sont transmises sur le bus en utilisant des protocoles spécifiques.

  • Les réseaux en étoile (téléphonie, STARLAN...)

    • Les réseaux en étoile:

    Les réseaux en étoile sont un type de réseau informatique dans lequel tous les périphériques sont connectés à un point central appelé concentrateur ou switch. La topologie en étoile est couramment utilisée dans les réseaux de téléphonie et dans les réseaux informatiques tels que STARLAN. Les données sont transmises de manière sélective entre les périphériques via le concentrateur.

  • Les réseaux en anneau:

    • Les réseaux en anneau (FDDI, Token-Ring):

    Les réseaux en anneau sont un type de réseau informatique dans lequel les périphériques sont connectés en une boucle fermée, formant un anneau. Les données sont transmises sur l'anneau dans un sens ou dans l'autre, en utilisant des protocoles tels que FDDI ou Token-Ring. Les données sont transmises de périphérique en périphérique jusqu'à ce qu'elles atteignent leur destination. Les réseaux en anneau offrent une meilleure résilience et tolérance aux pannes que les réseaux en bus.

  • Maillée:

    • Maillée (ATM, Réseaux MAN et WAN surtout):

    Internet est une topologie maillée (sur le réseau étendu « WAN » elle garantit la stabilité en cas de panne d'un nœud). Les réseaux maillés utilisent plusieurs chemins de transferts entre les différents nœuds. C'est une structure réseau hybride reprenant un câblage en étoile regroupant différents nœuds de réseaux.
