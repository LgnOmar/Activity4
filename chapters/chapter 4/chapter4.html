3 - Création d’une page web HTML

HTML : HyperText Markup Language est :

  • Un fichier texte, construit suivant une syntaxe précise
  • Structuré par des balises
  • Interprété (visualisé) dans un navigateur
  • Créé dans un éditeur de pages web

4 - Les principes de langage HTML

  • <head> </head>

    • Entête du document:
